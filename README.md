## 🌸 Project Overview
A premium digital wedding invitation experience crafted to celebrate the union of **Bhargav Raj** and **Sri Vaishnavi**.

Built with a modern web stack, this project blends traditional South Indian wedding aesthetics with elegant motion design, bilingual storytelling, and a luxurious guest experience.

More than an invitation, it is a memorable digital celebration for family and friends. ✨
